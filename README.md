Banking Database Star Schema
The project is designed using a **Star Schema** to model banking transactions. The central fact table stores transaction records, while customer, account, and branch information are maintained in dimension tables.

<img width="1408" height="768" alt="star_schema" src="https://github.com/user-attachments/assets/d5a76e3a-83f9-4c85-b3fe-de1092c2dd28" />

How to Run 
1. Execute 'database_schema.sql' to create the tables. 
2. Execute 'sample_data.sql' to insert sample data. 
3. Execute 'sql_queries.sql' to run the SQL practice questions.

About This Repository 
This repository showcases hands-on Oracle SQL practice using a Banking Management System database. It demonstrates SQL skills including joins, subqueries, window functions, aggregate functions, and business-oriented analytical queries, making it a portfolio project for Data Analyst.
